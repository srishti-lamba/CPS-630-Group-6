Iteration I & II

Instructions for running this application:
1. Copy the folder "Iteration1_2"
2. Inside the htdocs folder in XAMPP paste the folder ( "C:\xampp\htdocs\Iteration1_2" )
3. Then copy all the files into the folder
4. Turn on Apache and MySQL on the XAMPP Control Panel
5. On the XAMPP Control Panel, beside "MySQL" click "Admin"
6. On the left side bar of the browser page which opens, click "New"
7. Create a new database named "website" with a collation of "utf8mb4_general_ci"
8. Navigate to `http://localhost/React-ShoppingWebsite/Iteration1_2/views/home.php` on any browser to see the project
