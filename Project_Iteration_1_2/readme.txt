Instructions for running this application:
1. Inside the htdocs folder in XAMPP create a folder called "Project_Iteration_1_2"
2. Then copy all the files into the folder
3. Turn on Apache and MySQL on the XAMPP Control Panel
4. Navigate to `http://localhost/CPS-630-Group-6/Project_Iteration_1_2/views/home.php` on any browser to see the project
