Instructions for running this application:
1. Inside the htdocs folder in XAMPP create a folder called "Project_Iteration_1_2"
2. Then copy all the files into the folder
3. Turn on Apache and MySQL on the XAMPP Control Panel
4. On the XAMPP Control Panel, beside "MySQL" click "Admin"
5. On the left side bar of the browser page which opens, click "New"
6. Create a new database named "cps630" with a collation of "utf8mb4_general_ci"
7. Navigate to `http://localhost/CPS-630-Group-6/Project_Iteration_1_2/views/home.php` on any browser to see the project
